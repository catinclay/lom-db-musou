# 境界與連擊

> 相關：[merge](merge.md)（階級與忘形）、[combat](combat.md)（多波怎麼重選目標）。

`ComboTracker` 每回合持有兩個角色狀態：

- **境界**：突破門檻，起始 0。
- **連擊**：突破獎勵，起始 0。

出一張階級 R 的牌時：

- 若 `R > 境界`，境界與連擊各 +1，該牌使用增加後的連擊解算效果。
- 若 `R <= 境界`，連擊中斷，境界與連擊一起歸零；中斷牌仍以基礎倍率 `×1` 解算，但不沿用中斷前的連擊。

判定使用出牌前的境界。同階牌可以逐步把境界爬到該階級；到達後再出同階牌就會中斷。中斷後下一張有階級的牌從境界 0、連擊 0 重新起段。

忘形打出時只把境界歸零，連擊不變；因此低階牌可再次突破並把連擊往上續。

連擊的用途：

- **攻擊／純狀態牌**：`comboMultiplier(連擊)` 線性放大 `effect.hits`。暗器三發在連擊三時變九發；範圍招式會重打三波、每波重新選目標，崩山也逐波擊退。
- **功能牌**：與攻擊牌共用「連擊＝施放次數」。階級決定每次產量，連擊把整次效果重複。
  - 運氣調息：每次依階級回復 `3／4／5／6／7` 小格內力。
  - 臨機應變：每次依階級獲得 `3／4／5／6／7` 點靈感；每 3 點靈感換抽 1 張。
  - 例如四階臨機應變在連擊四時得到 `6 × 4 = 24` 靈感，因此抽 8 張，而不是把階級與連擊相乘成 16 次抽牌。

每回合 `reset()` 與連擊中斷都會將境界、連擊一起歸零。忘形本身沒有階級，不參與 `R <= 境界` 的中斷判定。
