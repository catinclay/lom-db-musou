## 里程碑 3「江湖遠征」— Phase 1–5 全上線

一天分成數個時辰，每個時辰「三選一」抽事件、入夜打尾王，殺戮尖塔式節奏推進到最終魔王；一局結束回門派據點花威望做永久升級。

- **Phase 1**：骨架 run loop（`RunState`＋`RunMapScene`＋有限戰鬥）。
- **Phase 2**：拉霸（`core/slot.js`＋`SlotScene`）、白天服務設施（`ShopScene`）、奇遇（`EventLibrary`＋`EventScene`）。
- **Phase 3**：遺物·秘籍（`RelicLibrary`）。
- **Phase 4**：主角屬性·階級上限（`RunState.attrs.maxRank`）。
- **Phase 5**：跨 run 門派據點 meta（`MetaState`＋`ui/metaStore.js`＋`BaseScene`）。

開機流程已擴充為 `TitleScene` 主題首頁 → `BaseScene` 七設施據點；據點包含成就、畫廊、跨局強化、
卡牌／事件／遺物總表與開始挑戰入口。成就與畫廊解鎖條件集中在 `ArchiveLibrary`。

詳細分層與流程見 [systems/run.md](systems/run.md)。

## 近期完成（2026-07-20）

- **主題首頁與據點設施化**：開機不再直接進門派據點；新增 `TitleScene` 開始畫面，據點拆成成就、畫廊、跨局強化、卡牌／事件／遺物總表與開始挑戰七個入口。
- **DoT 與狀態連擊修正**：燃燒回合末傷害為每層 1；毒霧／火藥層數吃階級，連擊改為多次獨立施放並錯開動畫。火藥後續爆炸不重複完全相同位置，且優先命中敵人。
- **敵人攻擊節奏**：接觸後依敵種進入黃色倒數，完成後顯示紅色攻擊意圖，再下一個敵方相位才攻擊；被推出攻擊線會重置準備。
- **敵種與特殊行動**：敵人擴充為嘍囉、快刀手、大漢、定樁力士；定樁力士能預告扎馬、停止前進並取得「不動」。不動會消耗層數抵銷擊退，也會阻斷後方推擠鏈。
- **補充波重構**：每波明確包含數排，只有成功入場才消耗。正常回合只補一排；玩家清場獲得 1 內力與 1 張牌，可按「再來啊！」一次叫進當前波剩餘排數。
- **資訊呈現**：敵人頭頂只顯示攻擊與特殊行動意圖；hover 顯示敵種、準備時間、下回合意圖及全部狀態／buff 說明。
- **驗證**：13 個測試檔、304 項測試全數通過；production webpack build 成功。

完成規格見 [plans/dot-status-scaling.md](plans/dot-status-scaling.md) 與 [plans/enemy-intents-and-wave-flow.md](plans/enemy-intents-and-wave-flow.md)。

## 近期完成（2026-07-21）

- **全場景順滑轉場**：首頁、據點、設施、日程、戰鬥、客棧、奇遇與拉霸的所有出口統一先淡入墨色再切換，目標畫面由墨色淡入；轉場中鎖輸入並防止重複觸發。
- **日內單位改稱時辰**：RunMap 以「第 N 時辰／尚餘 N 時辰」呈現同一天的推進，移除把事件數稱為「樁」的 UI 文案。
- **遊玩邏輯與 Phaser 完全解耦**：新增純 JS `GameSession` action／phase／snapshot 介面；地圖、奇遇、商店、拉霸、戰鬥 Scene 全改為只送 action 與重播結果。Node AI 可固定 seed、不開遊戲畫面高速反覆遊玩。
- **同步結算邊界**：回合結束一次算完 DoT、敵方相位與下一回合；拉霸在動畫前已套獎勵；戰後下一個 phase 只由 GameSession 決定。
- **非完全隨機時辰編排**：新增純 JS `OfferDirector`，用內部風險 pattern 保證每組至少一個安穩選項、內容不重複且功能有差異；近期內容降權，風險只靠文案暗示。
- **白天服務拆分**：客棧只供歇息、江湖商販賣招式／遺物、武館刪招、賭坊消耗代幣；沒代幣不出賭坊，付不起的消費設施降低權重。
- **有限低血救濟**：低於 30% 血量且沒有付得起的回血選項時，可能出現只回 15% 最大血量的山亭歇腳；每天至多一次、整局至多兩次。
- **驗證**：18 個測試檔、284 項測試全數通過；production webpack build 成功。

## 近期完成（2026-07-22）

- **連擊中斷**：有階級牌在 `rank <= 出牌前境界` 時中斷；境界與連擊一起歸零，中斷牌只以基礎倍率 ×1 結算。忘形無階級，不參與中斷判定。
- **卡牌去向劇本**：成功出牌由 core 明確產生 `DISCARD` 或 `EXHAUST`；棄牌飛往棄牌堆，消耗牌原地消散，UI 不再自行銷毀或推測去向。
- **忘形升階平衡**：忘形的打出與拖牌升階都會在本場消耗。拖牌升階可突破 `maxRank`，本身計為一次合成並給 2 點靈感，再接續同名同階自動合成鏈。
- **升階演出**：新增 `RANK_UP` transcript；忘形先消耗消散，目標以新 uid 原地升階，再播放補抽與後續合成。
- **靈感補牌**：移除合成的漸弱機率；每次合成／忘形升階給 2 點靈感，每滿 3 點必抽一張，餘數跨回合保留。每局初始靈犀玉讓每場開戰先有 2 點，保障前兩次合成都補牌。
- **功能牌統一**：運氣調息與臨機應變都採「連擊＝施放次數」；階級單次產量為 3／4／5／6／7，臨機應變改產生靈感。
- **內力細分**：3 小格組成 1 個完整氣輪，基礎回合內力改為 9 小格；所有舊制內力來源與費用等比換算，卡面與 HUD 共用大小格符號。
- **戰鬥提示降噪**：境界 0 時不高亮手牌；連擊 0 時不顯示右上連擊。靈感逐顆點亮並吃連鎖加速；第三顆滿溢後立即清空續跑，抽牌飛行與後續靈感並行。
- **驗證**：18 個測試檔、284 項測試全數通過；production webpack build 成功。
