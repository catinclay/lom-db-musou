## 里程碑 3「江湖遠征」— Phase 1–5 全上線

一天一輪輪「三選一」抽事件、入夜打尾王，殺戮尖塔式節奏推進到最終魔王；一局結束回門派據點花威望做永久升級。

- **Phase 1**：骨架 run loop（`RunState`＋`RunMapScene`＋有限戰鬥）。
- **Phase 2**：拉霸（`core/slot.js`＋`SlotScene`）、客棧商店（`ShopScene`）、奇遇（`EventLibrary`＋`EventScene`）。
- **Phase 3**：遺物·秘籍（`RelicLibrary`）。
- **Phase 4**：主角屬性·境界上限（`RunState.attrs`）。
- **Phase 5**：跨 run 門派據點 meta（`MetaState`＋`ui/metaStore.js`＋`BaseScene`）。

開機流程已擴充為 `TitleScene` 主題首頁 → `BaseScene` 七設施據點；據點包含成就、畫廊、跨局強化、
卡牌／事件／遺物總表與開始挑戰入口。成就與畫廊解鎖條件集中在 `ArchiveLibrary`。

詳細分層與流程見 [systems/run.md](systems/run.md)。

## 近期完成（2026-07-20）

- **主題首頁與據點設施化**：開機不再直接進門派據點；新增 `TitleScene` 開始畫面，據點拆成成就、畫廊、跨局強化、卡牌／事件／遺物總表與開始挑戰七個入口。
- **DoT 與狀態連段修正**：燃燒回合末傷害為每層 1；毒霧／火藥層數吃境界，連段改為多次獨立施放並錯開動畫。火藥後續爆炸不重複完全相同位置，且優先命中敵人。
- **敵人攻擊節奏**：接觸後依敵種進入黃色倒數，完成後顯示紅色攻擊意圖，再下一個敵方相位才攻擊；被推出攻擊線會重置準備。
- **敵種與特殊行動**：敵人擴充為嘍囉、快刀手、大漢、定樁力士；定樁力士能預告扎馬、停止前進並取得「不動」。不動會消耗層數抵銷擊退，也會阻斷後方推擠鏈。
- **補充波重構**：每波明確包含數排，只有成功入場才消耗。正常回合只補一排；玩家清場獲得 1 內力與 1 張牌，可按「再來啊！」一次叫進當前波剩餘排數。
- **資訊呈現**：敵人頭頂只顯示攻擊與特殊行動意圖；hover 顯示敵種、準備時間、下回合意圖及全部狀態／buff 說明。
- **驗證**：13 個測試檔、304 項測試全數通過；production webpack build 成功。

完成規格見 [plans/dot-status-scaling.md](plans/dot-status-scaling.md) 與 [plans/enemy-intents-and-wave-flow.md](plans/enemy-intents-and-wave-flow.md)。

## 近期完成（稀有度／絕學卡 ＋ 精英／魔王）

- **稀有度／絕學卡**：三階稀有度（普通/稀有/絕學），稀有度只影響取得、不改境界機制。稀有/絕學卡

  拉霸/郎中傳招、魔王戰利品皆走稀有度加權取得。新增「參悟武學」客棧服務（牌組某張境界永久 +1）。

  距離、召喚小兵、後退拉距，以及「每次出牌前進、可被打掉」的投射物。詳見
  [systems/combat.md](systems/combat.md) 的「精英／魔王」。
